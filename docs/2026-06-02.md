# 2026-06-02 — Tăng tốc mở lại cuộc họp (load-perf) + dọn data lỗi

## Mục tiêu phiên

Mở lại một cuộc họp đã lưu (cả **đang diễn ra** lẫn **đã xong**) hiện nội dung lên canvas **rất chậm** — dù scene đã nằm trong R2 và file đã có trong library. Lập **team agent** khảo sát toàn bộ path render để tìm gốc rễ, rồi sửa.

## Chẩn đoán — 3 lớp delay xếp chồng (team khảo sát)

1. **Scene (nét vẽ): chờ mù ~5 giây.** Thủ phạm chính. Lệnh tải scene từ R2 chỉ chạy *sau* khi socket bắt tay xong, qua sự kiện `first-in-room` hoặc khi peer gửi scene. Khi Leave rồi vào lại, socket cũ còn **ghost** trên room server (socket.io mặc định ~45s mới dọn) → server đếm room "chưa trống" → **không bắn `first-in-room`**, mà mình lại 1 mình nên không peer nào gửi → scene chỉ hiện sau **timeout dự phòng 5000ms** (`INITIAL_SCENE_UPDATE_TIMEOUT`).
2. **File/ảnh (IFC/DXF/PDF/ảnh):** `loadImageFiles` chạy sau scene render (trễ thêm); file 404 từ thời worker-down bị đánh dấu lỗi **vĩnh viễn** (`erroredFiles_fetch` làm `isFileTracked` = true → không fetch lại); peer fallback gửi dataURL nhiều MB **tuần tự** qua socket mã hoá → rất chậm cho IFC.
3. **Overlay MCM:** cache snapshot DXF/PDF/IFC nằm trong RAM, **reload là mất** → mỗi anchor DXF phải dựng lại WebGL renderer để bake lại PNG; overlay còn chờ hydrate thư viện (giải mã dataURL nhiều MB) mới chịu hiện.

## Đã sửa hôm nay (3 fix, typecheck pass)

1. **Eager parallel prefetch** — `Collab.tsx`: tải scene từ R2 **ngay lập tức, song song** với lúc socket connect; không chờ `first-in-room`/timeout 5s. Peer sống vẫn reconcile chồng lên sau (không set `socketInitialized` ở nhánh eager). Thêm cờ `eagerSceneLoaded` để `initializeRoom` bỏ qua re-fetch + resetScene (tránh nháy). → **Xoá hẳn 5s chờ** cho cả 2 kịch bản; file cũng bắt đầu tải sớm hơn ~5s theo.
2. **Reap ghost nhanh** — `room/src/index.ts`: `pingInterval 10s` + `pingTimeout 5s` (mặc định ~45s) → socket cũ rụng trong ~12s, `first-in-room` hoạt động đúng khi vào lại. **Cần restart room server :3002 để áp dụng.**
3. **Retry file tạm lỗi** — `storage.ts` `loadFilesFromStorage`: lỗi mạng/5xx (worker đang restart) thử lại 2 lần có backoff; **404 thật thì không retry** (khỏi phí thời gian với file chỉ có trên peer).
